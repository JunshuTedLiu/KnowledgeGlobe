Telling stories.
Intuitively.

Once we have the data, we can visualize it as graphs, which could tell you the results in an intuitive way, and even tell you stories in a selected time and space.

We are using R programming language for data visualization. With R, you can generate different kinds of graphs for your datasets, like bar plot, or rbokeh.

Learn more about R, one of the most wide-used programming language for data science.

Say you are asking...

"For which subject, researchers show more interests in the field of computer in the year 2013?"

We can easily visualize this in a bar plot:

As we can see, for all publications in the Springer Nature database about computers around the world in 2017, "computer science" and "artificial intelligence" are the most popular terms as subjects, and researchers show more interests on these as focuses.

We can even generate interactive graphs using rbokeh.

"For which country, researchers show more interests in the field of digital media in the year 2017?"

See this visualization in rbokeh. This is an interactive graph exported in HTML format. You can move your mouse on data points to see more details.

More examples are available in our GitHub's page.

Also, take a look at KnoGlo's R code to experiment on how to make these visualizations in action.
